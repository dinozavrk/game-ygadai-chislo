# Курсовой проект: Игра «Угадай число» с рекордами

Интерактивная информационная система, реализованная в виде веб-приложения. Программа загадывает случайное число в диапазоне от 1 до 100, а игрок должен угадать его за минимальное количество попыток. Система включает в себя полноценный функционал ведения таблицы рекордов.

 📋 Инструкция по запуску

1. Скачайте файлы проекта или клонируйте репозиторий командой 
```git clone git@github.com:dinozavrk/game-ygadai-chislo.git```
2. Откройте файл index.html в любом современном браузере (Chrome, Firefox, Edge, Safari).
3. Для сброса таблицы рекордов можно воспользоваться инструментами разработчика в браузере (Application -> Local Storage -> Clear).

 🚀 Функционал
- Игровой процесс: Генерация случайного числа (1–100) и система подсказок («Больше» / «Меньше» / «Вы угадали!»).
- Работа с пользователем: Персонализация игры (запрос имени перед началом).
- Таблица рекордов: 
    - Сохранение результатов (имя и количество попыток).
    - Автоматическая сортировка (от лучшего к худшему).
    - Ограничение списка до ТОП-10 лучших результатов.
- Сохранение данных: Использование localStorage позволяет сохранять таблицу рекордов даже после перезагрузки страницы или закрытия браузера.
- Интерфейс: Адаптивный (responsive) дизайн, корректная работа на мобильных устройствах и ПК.
- Валидация: Обработка некорректного ввода (пустые поля, буквы вместо чисел, отмена игры).

 🛠 Технологии
- HTML5: Семантическая разметка интерфейса.
- CSS3: Стилизация, обеспечение отзывчивости интерфейса.
- JavaScript (ES6+):
    - Модульная структура кода.
    - Работа с Web Storage API (localStorage).
    - Манипуляция DOM-деревом.

 ⚙️ Логика реализации
1. Архитектура: Код разбит на логические модули (функции управления состоянием игры, функции отрисовки интерфейса и функции работы с хранилищем).
2. Алгоритм сортировки: При каждом новом рекорде массив объектов обновляется, сортируется методом sort() по полю attempts и обрезается до 10 элементов.
3. UX/UI: Реализована возможность начать новую игру без перезагрузки страницы, что обеспечивает плавный пользовательский опыт.
